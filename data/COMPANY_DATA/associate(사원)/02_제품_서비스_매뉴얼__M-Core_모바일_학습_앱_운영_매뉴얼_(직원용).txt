작성일: 2025-08-19
회사: M-Core | 대상: 사원(일반 직원)
----------------------------------------------------------------------
M-Core 모바일 학습 앱 운영 매뉴얼 (직원용)

1) 개요
M-Core 모바일 학습 앱은 직원들이 언제 어디서나 학습할 수 있도록 지원하는 플랫폼입니다. 주요 시나리오는 다음과 같습니다:
- 직원 교육: 새로운 직원이 필수 교육 과정을 수료하는 시나리오.
- 자율 학습: 개인의 필요에 따라 자율적으로 학습 자료를 탐색하고 학습하는 시나리오.
- 성과 관리: 팀 리더가 팀원의 학습 진행 상황을 확인하고 피드백을 제공하는 시나리오.

2) 접근/권한
모바일 학습 앱에 접근하기 위해서는 최소권한 원칙을 준수해야 합니다. 모든 직원은 자신의 역할에 맞는 권한을 요청해야 합니다. 계정 요청 방법은 다음과 같습니다:
- 인사팀에 이메일로 요청.
- 요청 시 본인의 이름, 부서, 역할, 필요한 권한을 명시.
- 요청 후 2일 이내에 계정 생성 여부 확인.

3) 기본 사용 절차
다음은 모바일 학습 앱을 사용하는 기본 절차입니다:
1. 앱 다운로드: 앱 스토어에서 "M-Core 학습 앱" 검색 후 다운로드.
2. 로그인: 등록된 이메일과 비밀번호로 로그인.
   - 검증 포인트: 로그인 성공 여부 확인.
3. 과정 선택: 메인 화면에서 원하는 교육 과정을 선택.
   - 검증 포인트: 선택한 과정의 정보가 올바르게 표시되는지 확인.
4. 학습 시작: 과정에 포함된 자료를 학습.
   - 검증 포인트: 학습 진행 상태가 업데이트되는지 확인.
5. 완료 보고: 과정 완료 후 피드백 제출.
   - 검증 포인트: 피드백 제출 확인 메시지 수신.

4) 운영 체크리스트
일일 체크리스트:
- 로그인하여 새로운 알림 확인.
- 학습 진행 상황 점검.
- 팀원 학습 현황 확인 (팀 리더 전용).

주간 체크리스트:
- 완료한 교육 과정 검토.
- 미완료 과정 목록 작성.
- 다음 주 학습 계획 수립.

5) 문제 해결 FAQ
1. 앱이 다운로드되지 않아요. 어떻게 해야 하나요?
   - 기기의 저장 공간을 확인하고, 네트워크 연결 상태를 점검하세요.
2. 비밀번호를 잊어버렸어요. 어떻게 재설정하나요?
   - 로그인 화면에서 "비밀번호 찾기"를 클릭하고 지침을 따르세요.
3. 학습 진행 상황이 저장되지 않아요. 어떻게 해결하나요?
   - 앱을 재시작하거나 기기를 재부팅 후 다시 시도하세요.
4. 과정이 열리지 않아요. 무엇을 확인해야 하나요?
   - 앱 업데이트 여부와 인터넷 연결 상태를 확인하세요.
5. 피드백 제출이 안 돼요. 어떻게 해야 하나요?
   - 입력한 내용이 올바른지 확인하고, 다시 시도하세요.

6) 보안/컴플라이언스
모바일 학습 앱 사용 시 다음 사항을 준수해야 합니다:
- 개인 정보를 타인과 공유하지 않기.
- 공용 Wi-Fi 사용 시 VPN을 통해 접속하기.
- 앱 사용 중 의심스러운 활동이 발견되면 즉시 IT팀에 보고하기.
